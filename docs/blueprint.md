# **App Name**: TopTier Thrift Finds

## Core Features:

- Product Showcase: Display a grid of new arrivals and category-specific items with images, prices, and tags (Men/Women/Kids/Accessories).
- Detailed Item View: Present a modal/pop-up for each item showing a larger image, price, and condition/size information.
- Messenger Inquiry System: Allow users to 'Ask About This Item' directly via Facebook Messenger, facilitating personalized inquiries.
- Search & Filter: Enable users to search for items by keyword or filter them by category for easier discovery.
- Direct Contact Integration: Provide prominent WhatsApp and general Messenger contact buttons for broader inquiries, including bulk.
- Mobile-First Responsiveness: Ensure the application delivers an optimal viewing and interaction experience across all mobile devices, with fast loading times.

## Style Guidelines:

- Primary brand color: A deep forest green (#1B4D2E), selected for its eco-conscious feel and strong presence. This hue represents sustainability and quality.
- Background color: A very subtle, desaturated greenish-white (#F0F6F1), derived from the primary green, offering a clean, airy canvas for content that keeps the natural theme.
- Accent color: A versatile neutral gray (#666666) for secondary actions, text, and dividers, providing a professional and clean contrast.
- Headline and body font: 'PT Sans', a humanist sans-serif. It combines a modern aesthetic with warmth and readability, making it approachable for a diverse audience.
- Utilize a consistent set of clean, minimalist line icons to complement the modern design and maintain visual clarity on smaller screens.
- Adopt a mobile-first, grid-based layout for product display, prioritizing clear photography and essential information, designed to be easily digestible and fast-loading.
- Implement subtle and efficient micro-animations on interactive elements, ensuring a responsive feel without compromising fast load times for mobile users with slower internet connections.
